# Node.js 로컬 서버 실행 가이드

현재 근태관리 웹 애플리케이션은 직원 출퇴근 정보(`members.json`, `allmember.json`)를 서버에 영구적으로 기록하고 수정하기 위해 **Node.js 기반의 로컬 서버(`server.js`)** 구동이 필수적입니다.

서버를 실행하고 웹사이트에 접속하는 방법은 다음과 같습니다.

---

## 1. 사전 준비 (Node.js 설치)
- 컴퓨터에 Node.js가 설치되어 있어야 합니다.
- 설치 여부 확인법: 터미널(또는 명령 프롬프트) 창을 열고 `node -v`를 입력했을 때 버전 숫자(예: `v18.16.0`)가 나오면 정상 설치된 것입니다.
- 만약 설치되어 있지 않다면 [Node.js 공식 홈페이지(nodejs.org)](https://nodejs.org/)에서 다운로드 후 설치해 주세요.

---

## 2. 서버 실행 방법

1. **터미널(명령 프롬프트) 열기**
   - VSCode를 사용 중이시라면, 상단 메뉴에서 `Terminal` ➡️ `New Terminal` (또는 단축키 `` Ctrl + ` ``)을 엽니다.
   - 윈도우 기본 기능을 쓰신다면 `cmd`(명령 프롬프트)를 실행합니다.

2. **프로젝트 폴더로 이동**
   - 현재 홈페이지 소스코드(`server.js` 파일이 있는 곳)가 위치한 폴더로 이동합니다.
   - 예시 명령어: `cd c:\workspace\antig\test_web`
   - *(VSCode 터미널을 여셨다면 이미 해당 폴더 경로로 열려 있을 확률이 높습니다.)*

3. **서버 실행 명령어 입력**
   - 터미널에 아래의 명령어를 입력하고 엔터를 칩니다.
   ```bash
   node server.js
   ```

4. **실행 성공 확인**
   - 명령어 입력 후 터미널 창에 아래와 같은 메세지가 뜨면 서버가 정상적으로 켜진 것입니다.
   > `Server is running at http://localhost:8000`

---

## 3. 홈페이지 접속 방법
1. 크롬(Chrome)이나 엣지(Edge) 등 인터넷 브라우저를 엽니다.
2. 주소창에 아래 주소를 입력하고 접속합니다.
   ```text
   http://localhost:8000
   ```
3. 이제 로그인 화면이 나타나며, 출근/퇴근 및 관리자 기능이 정상적으로 파일에 저장됩니다.

---

## 4. 서버 종료 방법
- 홈페이지 사용(테스트)이 끝난 후 서버를 끄고 싶으시다면, 아까 켜두었던 터미널 창을 클릭하고 키보드에서 **`Ctrl + C`**를 누르면 서버가 즉시 종료됩니다.
